- MediatR is a popular .NET library that helps you implement the Mediator design pattern
  It allows components to communicate indirectly through a mediator instead of calling each other directly.



Instead of:
	controller -> service -> repository
You do:
	controller -> MediatR -> handler -> repository
	Controller -> MediatR (Command)  -> Handler -> Service -> Repository ->  Database
