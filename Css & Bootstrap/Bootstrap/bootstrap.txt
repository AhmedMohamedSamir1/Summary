<< bootstrap grid system >>

- by default each row takes 12 columns

- first of all : the parent element should be with class="row" --> display:flex; flex-wrap:wrap; // row>* {width:100%, max-width:100%}

- row children takes by default flex-shrink: 0; width: 100%; max-width: 100%;

- col -> elements with class col parent space divided equally between them, flex-grow: 1; flex-shrink: 0; flex-basis: 0% 

- col-6 -> child take 50% of parent width		

- You can control the number of columns per screen size:
	- row row-cols-1 row-cols-sm-2 row-cols-md-3 row-cols-lg-6
		// example 1
	        <div class="row row-cols-6">      // the div.row will have 6 column instead of 12
                    <div class="col-2">1</div>
                    <div class="col">2</div>
                    <div class="col">3</div>
                    <div class="col">4</div>
                    <div class="col">5</div>
                    <div class="col">6</div>
                </div>

	 	

                <div class="row row-cols-6">
                    <div class="col-6">1</div>  <!-- it takes 50% of parent which is 3 column -->
                    <div class="col">2</div>
                    <div class="col">3</div>
                    <div class="col">4</div>                    
                </div>


- the childs with class="col-md" --> the parent area will divided equally between childs on medium and larger screens(if not exist)
- the childs with class="col-sm" --> the parent area will divided equally between childs on small and larger screens
- xs --> this screen doesn't included in a class like the other screens, to determine it we can use this class --> col



- col-md-3 : on md screen and larger(if not exist) the child will take 3 columns from the row that contains 12 columns
- col-sm-3 : on sm screen and larger the child will take 3 columns from the row that contains 12 columns
- col-auto --> auto : child takes width that accomodate its conetnt only

                <div class="col-md-3 col-6">Ahmed</div>

<< Collapse >>

- .collapse : class indicates to this element is a collapsible element

- data-bs-toggle="collapse" : It tells Bootstrap that this element will be used to toggle (show/hide) a collapsible element (element with class collapse).

- href="#collapseExample" or data-bs-target="#collapseExample" : the collapsable element that will be toggled has the id collapseExample

- By default, the collapsible content is hidden. However, you can add the .show class to show the content by default

- when you click on the element that has data-bs-toggle="collapse" it will add class show to its target

- data-bs-parent="#sidebar" : make all collapsible elements under the specified parent will be closed when the collapsible item is shown.

--- Closing Alerts
	To close the alert message, add a .alert-dismissible class to the alert container. Then add class="btn-close" 
	and data-bs-dismiss="alert" to a link or a button element (when you click on this the alert box will disappear).

	- Example 1:
	  <div class="alert alert-success alert-dismissible">
  		<button type="button" class="btn-close" data-bs-dismiss="alert"></button>
  		<strong>Success!</strong> This alert box could indicate a successful or positive action.
	  </div>

	- Example 2 (Animated Alerts)
	  <div class="alert alert-danger alert-dismissible fade show">
		<button type="button" class="btn-close" data-bs-dismiss="alert"></button>
	  </div>
	   the alert disappears instantly when you click the close button.
	   fade → enables transition effect
	   show → makes the element visible and prepares it for animation
